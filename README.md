# Analisis-de-Ventas-y-Tendencias-en-la-Industria-de-los-Videojuegos
Descripción General
Este proyecto realiza un análisis exhaustivo de datos históricos de ventas de videojuegos desde 1980 hasta 2020. El estudio examina más de 16,000 registros para identificar patrones de mercado, comportamiento del consumidor y factores que influyen en el éxito comercial de los videojuegos. El análisis incluye la evolución temporal de lanzamientos, desempeño por plataforma, género, región geográfica y la relación entre calificaciones de críticos/usuarios con el rendimiento comercial.

# Objetivos
Identificar tendencias temporales: Analizar la evolución de lanzamientos y ventas a través de los años

Evaluar plataformas dominantes: Determinar qué consolas han liderado el mercado históricamente

Analizar preferencias por género: Identificar los géneros más populares y rentables

Examinar diferencias regionales: Comparar patrones de ventas en Norteamérica, Europa, Japón y otras regiones

Investigar impacto de calificaciones: Explorar la correlación entre puntuaciones de críticos/usuarios y éxito comercial

#Tecnologías Utilizadas

# Principales Hallazgos
Evolución del Mercado
Pico de lanzamientos: 2008-2009 con más de 1,400 juegos anuales

Declive posterior: Reducción significativa a partir de 2011

Plataformas líderes: PS2, X360, Wii y PS3 dominan en ventas totales

Distribución Geográfica
Norteamérica: Mercado principal para la mayoría de plataformas

Japón: Fuerte preferencia por plataformas locales y géneros específicos

Europa: Mercado consistente con tendencias similares a Norteamérica

# Factores de Influencia
Géneros dominantes: Acción, Deportes y Shooter muestran mejor desempeño

Calificaciones: Correlación positiva entre puntuaciones de críticos y ventas

Ciclo de vida de plataformas: Las ventas siguen patrones predecibles según la madurez de la consola

# Conclusiones
La industria muestra ciclos claros con periodos de crecimiento, saturación y consolidación

Las preferencias regionales son significativas y deben considerarse en estrategias de lanzamiento

La calidad (medida por calificaciones) influye en el éxito comercial, pero no es el único factor determinante

El análisis histórico proporciona insights valiosos para predecir tendencias futuras y orientar decisiones de desarrollo y marketing

La limpieza y preparación de datos es crucial para obtener conclusiones confiables, como demostró el proceso de imputación de años de lanzamiento faltantes

Este proyecto sirve como base para análisis predictivos más avanzados y estrategias data-driven en la industria del gaming.
